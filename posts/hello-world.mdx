# Tutorial: Hello, World! en JavaScript

En este tutorial, aprenderás cómo escribir tu primer programa "Hello, World!" en JavaScript. Este es un excelente punto de partida para familiarizarte con el lenguaje de programación JavaScript.

## Paso 1: Preparación del Entorno de Desarrollo

Antes de comenzar a escribir código, necesitarás tener un entorno de desarrollo configurado para JavaScript. A continuación se presentan los pasos generales:

1. **Instala un Navegador Web**: JavaScript se ejecuta en los navegadores web, por lo que necesitarás uno instalado en tu computadora. Google Chrome, Mozilla Firefox y Microsoft Edge son algunas opciones comunes.

2. **Editor de Código**: Elige un editor de código que te resulte cómodo para escribir JavaScript. Algunas opciones populares incluyen Visual Studio Code, Sublime Text y Atom.

## Paso 2: Creación del Archivo JavaScript

1. Abre tu editor de código y crea un nuevo archivo. Llámalo `hello.js`.

2. En este archivo, escribirás el código JavaScript para imprimir "Hello, World!" en la consola del navegador.

```javascript
// hello.js
console.log("Hello, World!");
```
